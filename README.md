# Tic Tac Toe Game

A simple **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.

This is my **first frontend project** where I combined HTML, CSS, and JavaScript together after learning JavaScript basics. The project helped me understand **DOM manipulation, game logic, event handling, and UI design**.

---

## 🚀 Features

* Interactive Tic Tac Toe gameplay
* Two-player mode
* Score tracking for both players
* Restart game button
* Reset scores button
* Rules section with close option
* Dynamic player marker switching (X/O)
* Clean and colorful UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

While building this project, I practiced:

---

## 🎮 Game Rules

1. X always starts first.
2. Players take turns.
3. Match 3 symbols in a row, column, or diagonal to win.
4. If all boxes are filled without a winner, the game is a draw.
